# Python学生成绩管理系统

## 核心功能
- 数据导出成csv文件
- 学生信息管理
- 课程信息管理
- 成绩信息管理
- 个人成绩单生成
- 成绩数据可视化

## 适用场景
- 教育机构
- 学校
- 教师个人使用

## 技术特点
- 使用Django框架进行开发
- 数据库采用MySQL
- 前端界面使用HTML、CSS和JavaScript构建
